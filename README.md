# WOWos

## An OS meant to be emulated in VirtualBox Or QEMU

You can get the floppy image for **QEMU** or the ISO image for **VirtualBox**

**DO NOT EXPECT MUCH!**

### What does this OS have?
> Typing,
> Mostly written in assembly,
> CTRL+Q To halt all operations,
> Very minimal

#### Notes
- This is not 32-bit or 64-bit.
- This is a 16-bit project, meaning that the size of this is very small.
- Shell uses hotkeys instead of commands(Kysh)
